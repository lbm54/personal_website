A comprehensive WordPress website built for the Unitarian Universalist Church of Birmingham, providing the congregation and community with information about services, events, and spiritual resources.

## About the Organization

The Unitarian Universalist Church of Birmingham is a welcoming faith community that embraces diverse spiritual backgrounds and promotes values of compassion, justice, and community engagement.

## Key Features

- **Service Information** - Details about weekly services, special events, and congregational gatherings.

- **Community Calendar** - Event listings and scheduling for church activities and programs.

- **Responsive Design** - Mobile-optimized layout ensuring accessibility across all devices.

- **Social Media Integration** - Connected with the church's social media presence for community engagement.

- **Easy Navigation** - Intuitive menu structure for visitors to find information about the church, its beliefs, and how to get involved.

## Technical Implementation

Built using WordPress with the Divi theme framework, enabling flexible page layouts and easy content management. The site uses custom typography with Roboto and Roboto Slab fonts for a clean, professional appearance. Staff can easily update content, add events, and manage church information through the WordPress admin interface.
